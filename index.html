Hello World

The "Hello, World!" program is a classic starting point in programming. It serves as a simple introduction to writing code and running a program. In Python, creating a "Hello, World!" program is quick and easy. Let's get started!

print("Hello, World!")

In the code above, we have a single line that prints the text "Hello, World!" to the console. Here's how you can run this program:

  1. Open a text editor or an Integrated Development Environment (IDE) of your choice.
  2. Create a new file and save it as hello_world.py, making sure to use the .py extension, which indicates that it's a Python file.
  3. Copy the code above into your hello_world.py file.
  4. Save the file.

Now, let's run the program:

  1. Open a command-line interface or terminal.
  2. Navigate to the directory where you saved your hello_world.py file.
  3. Execute the following command:
python hello_world.py

If everything is set up correctly, you should see the following output in the console:

Hello, World!
Variables

In Python, variables are used to store and manipulate data. They act as containers that hold values of different types, such as numbers, strings, or complex objects. Understanding variables is crucial as they play a central role in programming and enable you to work with and manipulate data effectively.

Naming Variables

When naming variables, there are a few rules and conventions to follow:

  • Variable names are case-sensitive, meaning myVariable and myvariable are considered different variables.
  • Variable names can consist of letters (both uppercase and lowercase), digits, and underscores (_). However, they must start with a letter or an underscore.
  • Avoid using reserved keywords, such as if, for, or while, as variable names.
  • Choose descriptive names that convey the purpose or meaning of the variable. For example, age, name, or total_sales are more meaningful than generic names like x or var1.
Variable Assignment

In Python, variables are assigned values using the = operator. For example: 


message = "Hello, World!"
count = 10
pi = 3.14159
Variable Usage

Variables can be used throughout your program. For instance: 


name = "Alice"
age = 25
greeting = "Hello, " + name + "!"
future_age = age + 5

Variables store data and can be updated or manipulated as needed.

Variable Reassignment

In Python, you can reassign new values to variables as needed. This flexibility allows you to update the value stored in a variable dynamically. Here's an example:


# Variable reassignment
x = 5
print(x)  # Output: 5

x = x + 1
print(x)  # Output: 6

x = x * 2
print(x)  # Output: 12

In the code above, we initially assign the value 5 to the variable x. We then reassign x to x + 1, effectively incrementing its value by 1. Finally, we multiply x by 2 and update its value once again.

Variable Scope

n Python, variable scope refers to the region or context in which a variable is defined and can be accessed. The scope determines where and for how long a variable exists during the execution of a program. Understanding variable scope is important for writing clean and organized code.

In Python, there are two main types of variable scope: global scope and local scope.

  1. Global Scope: Variables defined outside of any function or block have a global scope. They can be accessed from anywhere within the program, including inside functions. Global variables are typically defined at the top of the script or module and can be used throughout the entire program.

    Example of a global variable:

    
global_var = 10

def my_function():
    print(global_var) # Accessing the global variable
    my_function()  # Output: 10

  2. Local Scope: Variables defined inside a function or block have a local scope. They are only accessible within the function or block in which they are defined. Local variables exist as long as the function or block is being executed and are destroyed once the function or block finishes execution.

    Example of a local variable: 

    
  def my_function():
    local_var = 5
    print(local_var)  # Accessing the local variable

my_function()  # Output: 5

print(local_var)  # Error: NameError: name 'local_var' is not defined
Data Types

In Python, data types define the nature of the values that can be stored and manipulated within variables. Python provides several built-in data types that cater to different types of data and operations. Understanding data types is crucial as they determine the behavior and operations that can be performed on variables.

Here are some commonly used data types in Python:

  1. Numeric Types:
    • int: Integers, which represent whole numbers, e.g., 5, -10, 100.
    • float: Floating-point numbers, which represent decimal numbers with fractional parts, e.g., 3.14, -2.5, 1.0.
  2. Text Type:
    • str: Strings, which represent sequences of characters enclosed in single quotes ('') or double quotes (""). For example, "Hello, World!" or 'Python'.
  3. Sequence Types:
    • list: Lists, which are ordered and mutable collections of items. Lists can contain elements of different data types and can be modified after creation. tuple: Tuples, which are ordered and immutable collections of items. Tuples are similar to lists but cannot be modified once created. range: Ranges, which represent a sequence of numbers that can be iterated over. They are often used in looping constructs.
